Tablet Interaction Components

TabletViewer

Displays a tablet image with pan/zoom capabilities and optional interactive region overlays. The visual centerpiece of the Glintstone platform.

Live Example - Interactive viewer with zoom and regions

Interactive Behavior

Action Behavior
Click [-] Zoom out (step: 0.25x, min: 0.5x)
Click [+] Zoom in (step: 0.25x, max: 3.0x)
Scroll wheel + Ctrl Zoom at cursor position
Click and drag Pan when zoomed > 1.0x
Arrow keys Pan 10% of viewport when focused
Tab through regions Navigate interactive regions

Region States

State Border Background
default Transparent None
hover Solid, subtle 5% overlay
selected Solid, bold 15% overlay
ai-suggested Dashed, purple 5% AI overlay
Accessibility: Viewport is focusable with tabindex="0". Arrow keys pan when zoomed. Regions are buttons with descriptive aria-label. Screen reader announces tablet description on focus.

Variants

Variant Size Content Use Case
option 80-120px Image + name only Task options
compact 60px Image only Inline reference
default 120-160px Image + name + meaning Learning
detailed 200px+ Full information Sign library
Accessibility: When used as task options, wrap in role="radiogroup". Minimum touch target: 72x72px. Tab to navigate, Space to select.
