Layout Components

Container

Constrains content width and centers it within the viewport. Use for page-level content containment and readable line lengths.

Live Example - Resize to see behavior

Container content is centered and constrained.
Current: 1024px max-width

Variants

Variant Max Width Use Case
narrow 640px Text-focused content, forms
default 1024px Standard page content
wide 1280px Dashboard layouts
full 100% Edge-to-edge layouts

HTML Structure

<!-- Standard container -->
<div class="container">
  <main>Page content</main>
</div>

<!-- Narrow container for forms -->
<div class="container" data-size="narrow">
  <form>Form content</form>
</div>

Attribute Type Default Description

HTML Structure

<!-- Responsive card grid -->

  <article class="card">Card 1</article>

</div>

Sidebar

Creates a two-column layout with one flexible and one fixed column. Use for navigation + content, or tablet image + transcription.

Live Example - Side options

Sidebar

Fixed width area for navigation or secondary content.

Main Content

This area fills the remaining space. On small screens, the layout stacks vertically.

Props/Attributes

Attribute Type Default Description
data-side string "left" Which side for sidebar: left, right
data-width string "25%" Sidebar width
Accessibility: Use appropriate landmark elements: <aside> for sidebar, <nav> for navigation, <main> for primary content.

Switcher

Displays children in a row when space permits, stacking when constrained. Use for adaptive form layouts and button groups.

Live Example - Resize browser to see switching behavior

Form Fields (switches to stacked below ~480px)

Last Name

Stats Display

15 Tasks
3 Tablets
87% Accuracy

Behavior

  • Uses CSS container queries or viewport width
  • Switches between row and column layout automatically
  • All children have equal width in row mode
  • Content should make sense in both layouts

HTML Structure

<div class="switcher" data-threshold="480px">
  <div class="form-group">
    <label>First Name</label>
    <input type="text">
  </div>
  <div class="form-group">
    <label>Last Name</label>
    <input type="text">
  </div>
</div>
