Key Design Principles

Trust-First

Expert avatars, institutional logos, and credibility indicators integrated throughout to build user confidence in platform quality.

Accessibility-First

WCAG 2.1 AA baseline with AAA targets. Touch targets ≥44px, keyboard navigation, screen reader support, reduced motion.

Progressive Disclosure

Component complexity layers match user tiers (Passerby → Early Learner → Expert) without duplication.

Scalability

Clear path from JSON fixtures (R1) → SQLite → PostgreSQL + Redis + Elasticsearch (R4) with minimal refactoring.
